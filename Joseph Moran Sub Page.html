Github Hosting Tutorial

To start the process of hosting your website on GitHub make sure your html file is named index.html. This will let github know what file the main page is.

Next you need to create an account by going to github.com .


After your account is setup and you are logged in go to the top left of the screen, click the plus button and then select "New repository" from the dropdown.


Now you need to create a repository name, it can be anything you want. Before pressing create repository make sure to select the "Initialize this repository with README" option.


You now need to upload your file, again make sure the html file is named index.html, if it is not this will not work.


After uploading your file make sure "Commit directly to master branch" is selected and then click the green "Commit Changes" button.


Now when back on the main page for your repository navigate to settings.


Finally, scroll down to Github Pages in the options section of the settings. Then set source to master branch, select save, and you will be given a link for your hosted website. You will need to wait around a minute or so for this page to be processed.
